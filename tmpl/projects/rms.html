Проект: RMS (Resource Management System)

Web-приложение для планирования и управления загрузкой человеческих ресурсов при совместной работе над проектом. Приложение было сделано по заказу компании Webprofy для интеграции с популярным сервисом Teamwork . Основная цель приложения обеспечить удобный интерфейс для менеджера, чтобы он на одном экране видел целостную картинку загрузки всех участников его команды. А также имел возможность оперативно балансировать нагрузку перебрасывая задачи с более загруженных сотрудников на менее загруженных.

Данное web-приложение написано на AngularJS 1.3 и работает напрямую с сервером Teamwork через api данного сервиса. Никакой промежуточной сервер не используется.

Можно использовать данное приложение для своих задач. Для этого необходимо завести аккаунт на Teamwork (есть бесплатный тарифный план для нескольких проектов). Далее необходимо скопировать персональный токен из Teamwork и адрес вашего аккаунта. Видео инструкция по подключению прилагается.
