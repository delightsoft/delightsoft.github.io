Проект: RMS (Resource Management System)

Web-приложение для планирования и управления загрузкой человеческих ресурсов при командной работе над проектом. Приложение было сделано по заказу компании WebProfy для интеграции с популярным сервисом Teamwork .

Основная цель приложения - обеспечить удобный интерфейс для менеджера, чтобы он:

  • На одном экране видел целостную картину загрузки (занятости) всех членов его команды
  • Имел возможность оперативно выравнивать загрузку, перебрасывая задачи от более загруженных сотрудников к менее загруженным

Web-приложение написано на AngularJS 1.3 и работает напрямую с сервером Teamwork через api данного сервиса. Никакой промежуточной сервер не используется.

Можно ли использовать данное приложение для своих задач? Да, можно. Для этого необходимо завести аккаунт на Teamwork (есть бесплатный тарифный план для нескольких проектов). Далее необходимо скопировать персональный токен из Teamwork и адрес вашего аккаунта. Видео инструкция по подключению прилагается.
