Проект: Автопарк

У фармацевтической компании возникла потребность автоматизировать сбор и обработку отчетности по использованию корпоративного автотранспорта. Исходные данные:

  • Медицинские представители в различных городах России, за каждым из которых закреплен корпоративный автомобиль
  • Ежемесячно водитель сдает отчетность о пробеге и заправленном топливе
  • Ежемесячно водитель заполняет путевой лист, где расписывает маршруты своих поездок

Процесс был организован следующим образом. В компании были утверждены отчетные формы в MS Excel, которые пользователи корпоративных автомобилей заполняли в конце месяца и пересылали в транспортный отдел центрального офиса. Там эти формы обрабатывались вручную.

Бизнес-процесс включал следующие шаги:

  1. Сбор информации со всех пользователей автомобилями (процесс растягивался почти на месяц)
  2. Сверка расхода бензина с выписками по топливным картам (безналичный расчет) и с информацией из бухгалтерии (при заправке автомобилей за наличный расчет)
  3. Сведение всех данных в итоговый MS Excel файл и передача его в бухгалтерию

Цели разработки и внедрения системы Автопарк:

  • Повысить скорость обработки данных и устранить ошибки, неизбежные при ручной обработке excel файлов
  • Высвободить время медицинских представителей, чтобы они не тратили как минимум один день на заполнение отчетности по бензину, а использовали это время для основного вида деятельности: работе с врачами и аптеками
  • Организовать сбор информации в единую базу данных с последующим извлечением различной аналитической и статистической информации для руководства

На базе DSCommon было разработана и успешно внедрена система Автопарк, в которой была реализована следующая функциональность:

  • Личный кабинет для каждого водителя, через который сотрудник осуществляет следующие операции:
    • Ежемесячно вносит данные о пробеге
    • Вносит информацию об авариях
    • Отчитывается об оплате штрафов
    • Пишет заявки на ремонт (которые напрямую пересылаются в лизинговую компанию)
    • Заполняет заявки на покупку расходных материалов
  • Автоматическая рассылка напоминаний по email о необходимости заполнить ежемесячный отчет
  • Импорт данных из ПетролПлюс (поставщика топливных карт)
  • Импорт данных из 1С о заправках за наличный расчет
  • Автоматическая генерация путевых листов
  • Автоматическое формирование отчетов для бухгалтерии в формате для экспорта в 1С
  • Интеграция в корпоративную система - MS Active Directory (AD) авторизация

Реализация клиентских интерфейсов на web технологиях (по сути это обычные web-сайты), удалось подключить всех сотрудников (даже из самых удаленных городов России) без дополнительных усилий со стороны технических служб. Адаптивная верстка дизайна личного кабинета позволяет сотрудникам работать с информацией как с ноутбуков, так и с планшетов различных моделей.
