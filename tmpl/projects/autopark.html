Проект: Автопарк

У фармацевтической компании возникла потребность автоматизировать сбор и обработку отчетности по использованию корпоративного автотранспорта. Исходные данные:

  • Есть медицинские представители в различных городах России, за каждым из которых закреплен корпоративный автомобиль
  • Каждый месяц водитель сдает отчетность о пробеге и заправленном топливе
  • В конце месяца водитель заполняет путевой лист, где расписывает маршруты своих поездок

В компании процесс был организован следующим образом. Были утвержденные формы в MS Excel, которые заполнялись пользователями корпоративных автомобилей. Заполненные формы в виде файлов пересылались в центральный офис, в транспортный отдел, где вручную обрабатывались полученные файлы. Бизнес процесс состоял из следующих шагов:

  1. Собрать со всех сотрудников информацию (что само по себе растягивалось почти на месяц)
  2. Сверить расход бензина с выпиской по топливным картам и с информацией из бухгалтерии по заправкам автомобилей за наличный расчет
  3. Свести все данные в итоговый MS Excel файл и передать его в бухгалтерию

Цели разработки и внедрения:

  • Повысить скорость обработки данных и устранить ошибки, неизбежные при ручной обработке excel файлов
  • Высвободить время продавцов, чтобы они не тратили как минимум один день на заполнение отчетности по бензину, а использовали это время для основного вида деятельности: работе с врачами и аптеками
  • Сбор информации в единую базу данных с последующим извлечением различной аналитической и статистической информации для руководства

На базе DSCommon было разработана и успешно внедрена система Автопарк, в которой была реализована следующая функциональность:

  • Личный кабинет для каждого водителя, через который сотрудник осуществляет следующие операции:
    • Ежемесячно вносит данные о пробеге
    • Вносит информацию об авариях
    • Отчитывается об оплате штрафов
    • Пишет заявки на ремонт (которые напрямую пересылаются в лизинговую компанию)
    • Заполняет заявки на покупку расходных материалов
  • Автоматическая рассылка напоминаний по email о необходимости заполнить ежемесячный отчет
  • Импорт данных из ПетролПлюс (поставщика топливных карт)
  • Импорт данных из 1С о заправках за наличный расчет
  • Автоматическая генерация путевых листов
  • Автоматическое формирование отчетов для бухгалтерии в формате для импорта в 1С
  • Интеграция в корпоративную система - MS Active Directory (AD) авторизация

За счет того что клиентские интерфейсы были реализованы на web технологиях (по сути являлись обычными web сайтами) позволило без проблем подключить всех сотрудников (даже из самых удаленных городов России) без дополнительных усилий со стороны технических служб. Responsive верстка дизайна личного кабинета позволила сотрудникам работать с информацией как с ноутбуков, так и с планшетов различных моделей.
