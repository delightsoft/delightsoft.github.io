Проект: MyConomy - прототип обмена списками покупок

Стартап MyConomy разрабатывал мобильное приложение, позволяющее искать наилучшие цены на продукты в близлежащих магазинах. Для данного проекта была разработана функциональность, позволяющая обмениваться списками покупок между друзьями с онлайн синхронизацией активного списка. Например, компания собралась на шашлыки и закупается в крупном магазине по составленному списку. Синхронизация позволяет всем участникам процесса видеть одновременно только актуальную информацию - что еще не куплено.

Сделано:

  • сервер (play framework 1.2) отвечающий за синхронизацию списков между клиентами;
  • api для обмена данными между клиентом и сервером в формате json;
  • js-клиент на AngularJS;
  • верстка web-приложения;
  • возможность авторизации в системе через Facebook и Вконтакте, с возможностью делится списками с друзьями.
