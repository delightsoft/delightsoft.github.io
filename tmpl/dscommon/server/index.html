DSCommon Сервер

DSCommon (Data Service Common или DelightSoft Common) – библиотека находящая в разработке в компании DelightSoft. В настоящий момент разработка библиотеки находится в стадии завершения версии 1.0.

Библиотека работает на фреймворке Play Framework 1.3. Готова использовать в качестве БД практически любую SQL базу данных – достигается за счёт использование Hybernate. Основная БД на которой проходит тестирование – PostgreSql. Административный HTML интерфейс создан с активным использование Play! шаблонизации и библиотеки AngularJS.

Библиотека позволяет быстро задавать структуры данных, используя собственный формат описания документов в файлах на базе формата yaml. Структура из yaml файлов посредством кодогенератора переносится в java классы с hibernate аннтоцией – это значительно ускоряет процесс разработки Play! приложения. Позволяет избегать множества ошибок, возникающих при ручном аннотировании классов. В то же самое время информация из yaml файлов используется для задания прав ролей пользователей и для автогенерации элементов административного интерфейса.
